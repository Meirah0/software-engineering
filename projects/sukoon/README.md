# Sukoon – Digital Health System

Sukoon is a digital health application designed to improve patient experience, emotional support, and hospital navigation.

## Overview
The system focuses on reducing anxiety for patients and visitors by providing:
- Emotional support tools
- Clear hospital navigation
- Simple and user-friendly interface

## Project Structure

- gui-design/
  - README.md → UI explanation
  - report.md → Detailed design report

## Purpose
This project demonstrates the integration of healthcare management and computer science to build patient-centered digital solutions.
